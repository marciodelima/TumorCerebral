Introdução e Objetivo¶

O tumor cerebral uma das mais agressivas doenas em crianas, jovens e adultos. Os tumores cerebrais representam de 85 a 90% de todos os tumores primrios diagnosticados no Sistema Nervoso Central (SNC). Todos os anos, cerca de 11.700 pessoas so diagnosticadas com um tumor no crebro, a taxa de sobrevida em cinco anos de aproximadamente 34% para homens e 36% para mulheres.

Os tumores cerebrais so classificados como: Tumor benigno, Tumor maligno, Tumor hipofisrio, etc. A melhor tcnica para detectar tumores cerebrais a ressonncia magntica (RM).

Uma enorme quantidade de dados em imagens so geradas atravs das digitalizaes. Essas imagens so examinadas pelo mdico especializado.

Um exame manual est sujeito a erros humanos no diagnstico devido ao nvel de complexidades envolvidas nos tumores cerebrais e em suas propriedades. Desta forma, a aplicao de tcnicas de classificao usando Machine Learning (ML) e Deep Learning (Redes CNN) pode ser adotada e ajudar no diagnstico. Essa aplicao tem mostrado consistentemente maior preciso do que a classificao manual.

Informações sobre o DataSet¶

Este dataset contm imagens cerebrais de seres humanos no formato de JPG (MRI scan) categorizadas em 2 classes binrias:

  • NO - no temos um tumor. Tambm informado como 0;
  • YES - tumor encontrado. Tambm informado como 1.

No h maiores informaes sobre a origem dessas imagens no Kaggle.

O dataset contm 155 imagens com tumores e 98 imagens sem tumores. Desta forma, o dataset encontrasse balanceado. Os arquivos de imagens esto separados em diferentes diretrios (no e yes), desta forma, o diretrio representa seu label e target de identificao da resposta. O dataset apresenta base histrica, portanto nossos algoritmos sero do tipo de aprendizado supervisionado de classificao binria.

As imagens esto no formato JPG, mas esto em formato de pixels diferentes, variando de 200 pixels a 380 pixels de algura e largura. Esto em formato preto e branco com 8 bits. Os arquivos so datados no ano de 2019.

Pre-processamento¶

Existem vrias formas de tratar os arquivos, eu escolhi devido a facilidade e meu conhecimento prvio de juntar os arquivos e seus labels para dividir melhor os dados para treino e teste futuro.

Os arquivos serão lidos e importados com tamanho 128 x 128 pixels. Esse resizing tem o objetivo de padronizar os arquivos para o mesmo tamanho em pixels para a correta aprendizagem dos modelos e para ter mais performance no modelo.¶

As imagens do MRI j esto no padro de cor correto, desta forma, no houve necessidade de transformar => 8-bit pixels, black and white

Utilização da técnica de Normalização, deixando todos os valores no intervalo entre 0 e 1. Desta forma, ganhamos performance no treinamento da rede neural e aumentamos a acurária.¶

Criando os modelos¶

Para este trabalho escolhi o SVC, XGBoost e Redes Neurais com o Keras.

Escolhido o mtodo de Cross-Validation para os treinamentos dos modelos. O Cross Validation uma tcnica que pode ser utilizada para avaliar a performance de um modelo com menos varincia que a tcnica de dividir os dados em treino/teste. Confome estudo em Kuliahkomputer (2018), com esta tcnica dividimos os dados em partes normalmente chamadas de k-folds (por exemplo k = 5, k = 10). Cada parte chamada de fold. Cada fold usado no treinamento de forma repetida e um fold por vez. Aps executar o processo em k-1 folds, podemos sumarizar a performance em cada fold usando a mdia e o desvio padro. O resultado normalmente mais confivel e oferece maior acurcia ao modelo (Kuliahkomputer, 2018).

Os modelos XGBoost e SVC foram definidos com hiper-parmetros em seu default pois so a primeira verso de modelo.

#Criando e compilando a Rede Neural CNN - Keras
def build_model():
    model_cnn = Sequential()
    model_cnn.add(Conv2D(32, kernel_size=(3, 3), activation='relu', input_shape=(X.shape[1],X.shape[2],X.shape[3])))
    model_cnn.add(MaxPooling2D(pool_size=(2, 2)))
    model_cnn.add(Conv2D(64, kernel_size=(3, 3), activation='relu'))
    model_cnn.add(MaxPooling2D(pool_size=(2, 2)))
    model_cnn.add(Dropout(0.3))
    model_cnn.add(Flatten())
    model_cnn.add(Dense(128, activation='relu'))
    model_cnn.add(Dropout(0.5))
    model_cnn.add(Dense(units=1, activation='sigmoid'))
    model_cnn.compile(optimizer='rmsprop', loss='binary_crossentropy', metrics=['accuracy'])
    return model_cnn

As funes de otimizao, erro e de ativao da ltima camada (sada) foram escolhidas devido ao caso de uso (Classificao binria (1 sada)). Se o caso de uso fosse de Multi-Classes seriam escolhidos outros parmetros.

Utilizado o Dropout para evitar o overfitting, usando 1 de 30% na primeira vez e 1 mais rgido de 50% no final.

Utilizado o kernel_size de 3 x 3 pixel como tamanho da janela de filtro da Conv2D.

Escolhida a funo de ativao RELU devido ser a mais utilizada e comum para a grande maioria dos casos de uso.

As camadas de filtro iniciam em 32 (definido como padro inicial) e dobram de valor a cada camada, indo pra 64 e depois 128. Foi utilizado filtro de 32 devido a importncia do caso de uso em ter a maior acurria possvel, pois tratasse de diagnstico de sade humana.

Escolhido o algoritmo de otimizao Root Mean Square Propogation (RMSProp) proposto por Geoffrey Hinton. Esse otimizador busca atenuar as oscilaes verticais e elimina a necessidade de ajustar a taxa de aprendizado.

Escolhido a tcnica de Max pooling para reduo de dimensionalidade a cada camada, essa tcnica extrai os valores mais altos de cada janela (window). O Max pooling a cada camada proporciona melhor desempenho no treinamento do modelo.

#Cross-Validation de Redes Neurais
def k_fold_train(model, x_train, y_train):
    kfold = StratifiedKFold(n_splits=num_folds, shuffle=True, random_state=seed)
    k_fold_accuracies = []

    for k_train, k_test in kfold.split(x_train,y_train):
        history = model.fit(
            x_train[k_train],
            y_train[k_train],
            epochs=10,
            batch_size=20,
            verbose=0,
            validation_data=(x_train[k_test], y_train[k_test]))

        #Acuraria
        score = history.history['val_accuracy']
        k_fold_accuracies.append(score)

    return k_fold_accuracies

def build_model_cnn(camadas=[32,64,128], tamanho=(128,128,3), activate='relu', drop=0.5, var_optimizer='adam', l2_penalty=0.01):
    model_cnn = Sequential()
    model_cnn.add(Conv2D(camadas[0], kernel_size=(3, 3), activation=activate, input_shape=tamanho, kernel_regularizer = l2(l2_penalty)))
    model_cnn.add(MaxPooling2D(pool_size=(2, 2)))
    model_cnn.add(Conv2D(camadas[1], kernel_size=(3, 3), activation=activate, kernel_regularizer = l2(l2_penalty)))
    model_cnn.add(MaxPooling2D(pool_size=(2, 2)))
    model_cnn.add(Dropout(drop/2))
    model_cnn.add(Flatten())
    model_cnn.add(Dense(camadas[2], activation=activate, kernel_regularizer = l2(l2_penalty)))
    model_cnn.add(Dropout(drop))
    model_cnn.add(Dense(units=1, activation='sigmoid'))
    model_cnn.compile(optimizer=var_optimizer, loss='binary_crossentropy', metrics=['accuracy'])
    return model_cnn
